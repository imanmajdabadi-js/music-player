# Music Player Web App

A web-based music player demonstrating component-based architecture and React state management.

## Features

- Play / Pause audio
- Next / Previous track navigation
- Playlist with active track highlighting
- Progress bar with seek functionality
- Responsive user interface

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
